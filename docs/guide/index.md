# Get Started  

## Overview  

Qr grid is a powerful, customizable JavaScript library for generating QR codes. Itâ€™s designed to be lightweight, with no external dependencies, and supports rendering in both Canvas and SVG formats.

## Installation

### Core

Installation of the Qr grid library for Web applications.

::: code-group

```sh [npm]
npm install @qrgrid/core
```

```sh [bun]
bun add @qrgrid/core
```

```sh [pnpm]
pnpm install @qrgrid/core
```

```sh [yarn]
yarn add @qrgrid/core
```

:::

For more details, see the [documentation](./packages/core).
